# Veo3 CLI

A comprehensive command-line utility for Google's Veo 3.1 video generation API.

## Installation

```bash
go install github.com/jasongoecke/go-veo3/cmd/veo3@latest
```

## Usage

```bash
veo3 generate "A cinematic shot of a majestic lion in the savannah"
```

## Configuration

Run `veo3 config init` to set up your API key and defaults.

## Development

```bash
make build
make test
```
